# Crystal UI

Lightweight CSS library for building modern web interfaces with pure HTML and CSS. No React or JavaScript framework required.

## Installation

Include the CSS file in your HTML:

```html
<link rel="stylesheet" href="https://crystal-ui.com/v1/crystal-ui.css" />
```

Or download it locally:

```bash
# Download from CDN
curl -o crystal-ui.css https://crystal-ui.com/v1/crystal-ui.css
```

## Quick Start

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <link rel="stylesheet" href="https://crystal-ui.com/v1/crystal-ui.css" />
  </head>
  <body>
    <button class="cui-btn cui-btn--primary">Click me</button>
  </body>
</html>
```
